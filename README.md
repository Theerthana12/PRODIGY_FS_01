
# PRODIGY_FS_01 - Secure User Authentication System

## Features
- User Registration
- Secure Login
- Password Hashing (bcrypt)
- JWT Authentication
- Protected Dashboard
- Logout functionality

## Tech Stack
Frontend: React (Vite)
Backend: Node.js, Express
Database: MongoDB
Authentication: JWT + HTTP-only Cookies

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

Make sure MongoDB is running and .env file is configured.
